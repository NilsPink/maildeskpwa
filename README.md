# MailDesk PWA

AI-powered email reactie generator voor Pink Roccade Servicedesk.

**Functies:**
- ✨ AI Builder met chips
- 📋 26+ Templates
- 📋 PCR Maker
- 🔍 Analyse & Samenvatting
- 📅 Operationeel Overleg
- 🌙 Dark Mode
- 📱 Offline Capable

**Link:** https://NilsPink.github.io/maildeskpwa

## Setup via GitHub Pages

Zie SETUP.md voor lokale setup.

## Features

### AI Builder
Selecteer chips → AI genereert reactie → per alinea alternatieven

### Templates
26 kant-en-klare templates voor alle situaties

### PCR Aanmaken
Plak melding → AI maakt story (5-15 regels) + taak (max 2 zinnen)

### Analyse
Samenvatting, advies, volledigheidscheck, vervolgvragen

### Dark Mode
🌙 Toggle rechts in header

## Offline

App werkt offline voor:
- UI navigatie ✓
- Templates ✓
- Afbeeldingen ✗ (vereist internet)
- AI genereren ✗ (vereist internet + API)

## Installeren als App

1. Open de link in Chrome/Edge
2. Klik **+** in URL-balk
3. Klik **Installeren**
4. App verschijnt in Start Menu / App Drawer
